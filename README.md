## Descrição

[whatsapp-api](https://github.com/andersonaguia/whatsapp-api) é uma API Rest para envio de mensagens via whatsapp utilizando a biblioteca [venom-bot](https://github.com/orkestral/venom).

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
